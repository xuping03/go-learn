# 22. map

## 创建

```go
var m1 map[int]string            // 声明，值为 nil，不能直接赋值
m2 := make(map[int]string)       // 初始化
m3 := make(map[int]string, 2)    // 初始化，预分配容量
```

## 增删查

```go
m3[1] = "a"         // 赋值
delete(m3, 1)       // 删除

value, ok := m3[1]  // 查，ok=false 表示键不存在
```

## 遍历

```go
for key, value := range m3 {
    fmt.Println(key, value)
}
```

## 关键点

- map 是引用类型，作为参数传递时函数内修改会影响原 map
- map 的遍历顺序不固定
- 并发读写 map 不安全，需要加锁或使用 `sync.Map`
