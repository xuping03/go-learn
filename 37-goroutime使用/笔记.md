# 37. goroutine 并发

## 基本用法

```go
go newTask()   // 启动新 goroutine，与 main 并发执行

go func() {   // 匿名函数 goroutine
    for i := 0; i < 5; i++ {
        fmt.Println("go")
    }
}()
```

## runtime 常用函数

```go
runtime.Gosched()         // 让出当前时间片
runtime.Goexit()          // 退出当前 goroutine（defer 仍执行）
runtime.GOMAXPROCS(n)     // 设置并行 CPU 核数
```

## 关键点

- 主 goroutine 退出则所有子 goroutine 也退出
- `Goexit()` 只退出当前 goroutine，不影响其他
- 默认 `GOMAXPROCS = CPU 核数`，Go 1.5+ 自动设置
