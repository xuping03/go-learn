# 24. 匿名字段（嵌入）

## 定义

```go
type Person struct {
    name string
    age  int
}

type Student struct {
    Person        // 匿名字段（嵌入 Person）
    school string
    id     int
}
```

## 初始化

```go
s := Student{Person{"mike", 20}, "qinghua", 123}
s2 := Student{Person: Person{name: "mike"}}
```

## 多个匿名字段

```go
type mystr string
type Student2 struct {
    Person
    int
    mystr
}
s := Student2{Person{name: "xx"}, 12, "tt"}
```

## 关键点

- 匿名字段将外层类型的字段和方法**提升**到当前结构体，可直接访问
- 同名字段时，外层字段优先，若需访问内层需显式 `s.Person.name`
- `%+v` 格式打印可显示字段名
