# 08. 函数

## 基本定义

```go
func myFun(a int) { }
```

## 可变参数

```go
func myFun2(args ...int) {
    for i, v := range args { }
    myFun3(args...)      // 传递全部
    myFun3(args[2:]...)  // 从第 3 个开始
    myFun3(args[:2]...)  // 前两个（不含 args[2]）
}
```

## 返回值

```go
// 直接返回
func f(a int) int { return a + 1 }

// 命名返回值（推荐）
func f(a int) (result int) {
    result = a + 1
    return
}

// 多返回值
func f() (a int, b int, c int) {
    a, b, c = 11, 22, 33
    return
}
```
