# 23. 结构体

## 定义与初始化

```go
type Student struct {
    id   int
    name string
    age  int
}

// 顺序初始化
s1 := Student{1, "xiaoming", 18}

// 指定字段初始化（推荐，未初始化的字段为零值）
s2 := Student{name: "xiaohong"}
```

## 指针结构体

```go
p2 := &Student{name: "mike"}      // 指针字面量
p3 := new(Student)                // new 分配
p3.id = 1                         // 通过指针访问字段
(*p3).name = "linux"              // 等价写法
```

## 关键点

- 结构体是**值类型**，赋值或传参时完整拷贝
- 通过指针修改字段，`p.field` 和 `(*p).field` 等价
- 字段首字母大写才能被其他包访问
