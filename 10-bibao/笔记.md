# 10. 闭包

## 概念

闭包是一个函数 + 其捕获的外部变量。与普通函数的区别：**闭包中捕获的变量在多次调用间保持状态**。

## 示例

```go
func test02() func() int {
    var x int
    return func() int {
        x++
        return x * x
    }
}

f := test02()
fmt.Println(f()) // 1（x=1, 1*1）
fmt.Println(f()) // 4（x=2, 2*2）
```

对比普通函数每次调用 x 都从 0 开始，闭包的 x 跨调用保持。

## 关键点

- 闭包捕获的是变量的**引用**，不是值的副本
- 每次调用 `test02()` 都会创建一个新的独立闭包和独立的 x
