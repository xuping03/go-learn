# 05. switch 语句

## 基本用法

```go
switch num {
case 1:
    fmt.Println("一楼")
case 2:
    fmt.Println("二楼")
default:
    fmt.Println("其他")
}
```

## 关键点

- Go 的 switch **默认有 break**，不会自动 fallthrough
- 使用 `fallthrough` 可强制继续执行下一个 case

```go
case 1:
    fmt.Println("一楼")
    fallthrough  // 继续执行 case 2
case 2:
    fmt.Println("二楼")
```

## 带初始化语句

```go
switch c := 1; c {
case 1: // ...
}
```

## 无条件 switch（替代多条件 if-else）

```go
switch {
case score >= 90: fmt.Println("优秀")
case score >= 80: fmt.Println("良好")
default:          fmt.Println("不及格")
}
```
