# 25. 面向过程 vs 面向对象

## 面向过程（函数操作数据）

```go
func printInfo(name string, age int) {
    fmt.Println(name, age)
}
```

## 面向对象（方法绑定到类型）

```go
type Person struct { name string; age int }

func (p Person) PrintInfo() {
    fmt.Println(p.name, p.age)
}
```

## 区别

- 面向过程：数据和操作分离，函数接收数据作为参数
- 面向对象：数据和操作绑定在一起，方法通过接收者操作自身数据
