# 27. 方法的继承

通过匿名字段（嵌入），子类型自动**继承**父类型的所有方法：

```go
type Person struct { name string; sex byte; age int }

func (tmp *Person) PrintInfo() {
    fmt.Printf("name=%s, sex=%c, age=%d\n", tmp.name, tmp.sex, tmp.age)
}

type Student struct {
    Person  // 嵌入 Person
    id int
}

s := Student{Person{"张三", 'g', 19}, 1}
s.PrintInfo()  // 直接调用 Person 的方法
```

## 方法重写

子类型可定义同名方法覆盖父类型：

```go
func (s *Student) PrintInfo() {
    fmt.Println("Student:", s.name)
    // 调用父类方法：s.Person.PrintInfo()
}
```
