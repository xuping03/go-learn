# 21. 猜数字

综合练习：随机数 + 循环 + 条件判断 + 控制台输入

```go
rand.Seed(time.Now().UnixNano())
target := rand.Intn(100)

for {
    var guess int
    fmt.Scan(&guess)
    if guess > target {
        fmt.Println("猜大了")
    } else if guess < target {
        fmt.Println("猜小了")
    } else {
        fmt.Println("猜对了")
        break
    }
}
```
