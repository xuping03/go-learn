# 15. 数组

## 声明与初始化

```go
var a [5]int = [5]int{1, 2, 3, 4, 5}
b := [5]int{6, 7, 8, 9, 10}
c := []int{1, 2, 3}         // 切片（非数组）
d := [5]int{1, 2, 3}        // 未初始化部分自动为 0
```

## 二维数组

```go
e := [3][4]int{1: {5, 6, 7, 8}}  // 只初始化第 2 行
```

## 比较

```go
e1 := [2]int{1, 2}
e2 := [2]int{1, 2}
fmt.Println(e1 == e2) // true，逐元素比较
```

## 关键点

- 数组长度是类型的一部分，`[3]int` 和 `[5]int` 是不同类型
- 数组作为函数参数是**值传递**（拷贝），修改不影响原数组
- 如需修改原数组，传指针或使用切片
