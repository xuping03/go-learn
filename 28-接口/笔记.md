# 28. 接口

## 定义接口

```go
type Humaner interface {
    sayHi()
}
```

## 实现接口

只要实现了接口的全部方法，就隐式满足接口（无需显式声明）：

```go
type Student struct { name string; age int }

func (tmp *Student) sayHi() {
    fmt.Println("student say hi", tmp)
}

type Teacher struct { name string; address string }

func (tmp *Teacher) sayHi() {
    fmt.Println("teacher say hi", tmp)
}
```

## 使用接口（多态）

```go
var i Humaner

i = &Student{"mike", 6}
i.sayHi()

i = &Teacher{"bj", "go"}
i.sayHi()
```

## 关键点

- 接口只定义行为（方法签名），不包含数据
- Go 接口是隐式实现，无 `implements` 关键字
- 空接口 `interface{}` 可以接受任意类型
