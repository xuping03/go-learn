# 19. 切片

## 创建方式

```go
s := make([]int, 5, 10)   // 长度 5，容量 10
s := []int{1, 2, 3}       // 字面量
s := arr[0:3:5]           // 从数组切割，[low:high:cap]
```

## 常用操作

```go
len(s)          // 长度
cap(s)          // 容量
s = append(s, 11)  // 追加元素，容量不足时自动扩容
```

## 切片再切片

```go
a := [9]int{1,2,3,4,5,6,7,8,9}
s3 := a[2:5]    // [3 4 5]，len=3
s4 := s3[2:7]   // 可以超过 s3 的 len，但不能超过 cap
```

## 关键点

- 切片是对底层数组的引用，多个切片可能共享同一底层数组
- `append` 超出容量时会分配新数组，与原数组脱离关联
- `[low:high:cap]` 第三个参数限制新切片的容量
