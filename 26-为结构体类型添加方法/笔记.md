# 26. 为结构体类型添加方法

## 值接收者（不修改原值）

```go
func (tmp Person) PrintInfo() {
    fmt.Println(tmp)
}
```

## 指针接收者（修改原值）

```go
func (p *Person) SetInfo(n string, s byte, a int) {
    p.name = n
    p.age = a
}
```

## 调用

```go
p := Person{"张三", 20, 'm'}
p.PrintInfo()
p.SetInfo("李四", 22, 'f')  // Go 自动取地址
```

## 同名方法

不同接收者类型可以定义同名方法，不冲突：

```go
func (tmp long) test()  {}
func (tmp char) test()  {}
```

## 关键点

- 接收者名称通常用类型首字母小写（惯例）
- 需要修改数据或避免拷贝开销时，用指针接收者
- 方法调用时 Go 会自动在值和指针之间转换
