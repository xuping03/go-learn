# 29. 类型断言

## if 方式（安全断言）

```go
i := make([]interface{}, 3)
i[0] = 1
i[1] = "hello go"
i[2] = Student{"mike", 666}

for index, data := range i {
    if value, ok := data.(int); ok {
        fmt.Println(index, value)  // 只处理 int 类型
    }
}
```

## switch 方式（类型选择）

```go
switch v := data.(type) {
case int:
    fmt.Println("int:", v)
case string:
    fmt.Println("string:", v)
default:
    fmt.Println("other type")
}
```

## 关键点

- `value, ok := x.(T)`：安全断言，ok=false 时不 panic
- `value := x.(T)`：不安全断言，断言失败直接 panic
- 类型断言只能用于接口类型变量
