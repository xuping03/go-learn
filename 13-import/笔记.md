# 13. import 包导入

## 基本用法

```go
import (
    "fmt"
    "go-learn/13-import/otherfun"  // 导入自定义包
)

otherfun.Add(1, 2)    // 调用包内导出函数（首字母大写）
otherfun.Minus(3, 1)
```

## 导出规则

- 包内**首字母大写**的函数/变量/类型才能被其他包访问
- 首字母小写只能在同一包内使用

## 导入别名

```go
import f "fmt"   // 别名导入
f.Println("hi")

import . "fmt"   // 点导入，可直接使用函数名（不推荐）
Println("hi")

import _ "pkg"   // 只执行 init()，不使用包
```
