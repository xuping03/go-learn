# 11. defer

## 基本用法

```go
defer fmt.Println("bbb")
defer fmt.Println("aaa")
defer fmt.Println("ccc")
// 输出顺序：ccc → aaa → bbb（后进先出，LIFO）
```

## 关键点

- `defer` 在函数返回时执行，多个 `defer` 按**后进先出（栈）**顺序执行
- `defer` 语句的参数在声明时就被求值，而非执行时
- 常用于资源释放：`defer file.Close()`、`defer mutex.Unlock()`
- 即使函数发生 panic，`defer` 仍然执行
